Display a Phylogenetic Tree with a Circular Dendrogram Style
============================================================


.. code-block:: default

    from cogent3.app import io


    reader = io.load_json()

    ens_tree = reader("../../data/GN-tree.json")
    fig = ens_tree.get_figure("circular", width=600, height=600)
    fig.show()
